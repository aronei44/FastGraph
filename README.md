# FastGraph
Base project with FastAPI and GraphQL (Strawberry)

## Installation

- clone this repository
```
git clone https://github.com/aronei44/FastGraph.git
cd FastGraph
```
- virtualenv (win)
```
py -m venv venv
./venv/Script/activate
pip install -r requirements.txt
```
- run app (win)
```
py main.py
```
